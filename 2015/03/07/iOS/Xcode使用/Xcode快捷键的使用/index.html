Xcode快捷键的使用

发表于 2015-03-07

快捷键在Xcode的工具栏里都标注有，下面导航部分可以看出。

  • 运行调试

    Command + R  运行。
Command + .  停止
F6单步调试、F7跳入，F8继续

    • 导航

      下面标识一些符号和对应键

      常用快捷键
      Command + control+ 上下箭头键 切换头文件和m.文件
      control+command+ 左箭头 go back，回到你上次打开的页面
      control+command+ 右箭头 go forward 回到你前面打开的页面

  • 文件

    command+shift+n  新建项目 
command+n  新建文件 
command+control+n  新建空文件 
command+o  打开 
command+w  关闭窗口  
command+option+s  保存所有文件
command+u  还原到保存时状态

  • 编辑代码

    • 代码缩进

      command+[  左缩进 
command+]  右缩进

    • 代码编辑

      control-F  向右一个字符（forward）
control-B  向左一个字符（backward）
control-P  前一行（previous）
control-N  后一行（next）
control-A  去行首
control-E  到行尾（end）
control-T  调换光标两边的字符（transpose）
control-D  删除光标右侧字符（delete）
control-K  删除本行剩余的字符（kill）
Command+<-  删除本行

      • 文档内搜索

        Command + F: 搜索
        Command + G: 搜索下一处
        Shift + Command + G: 搜索上一处
